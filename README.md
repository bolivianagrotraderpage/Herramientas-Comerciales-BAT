# Herramientas Comerciales

Calculadora de descuento por humedad e impureza para Soya, Maíz y Sorgo.

## Contenido

- `comercial.html` — página única, sin dependencias externas (aparte de fuentes de Google Fonts). Todo el cálculo ocurre en el navegador, no requiere backend ni Google Sheets.

## Uso

Abrir `comercial.html` directamente en un navegador, o publicarlo con GitHub Pages (Settings → Pages → branch `main` / carpeta raíz).

## Detalles de cálculo

- **Soya**: referencia Humedad 13% / Impureza 1%. Precio y peso en **USD / Tonelada Métrica (TM)**.
- **Maíz** y **Sorgo**: referencia Humedad 14% / Impureza 1%. Precio y peso en **Bs / quintal (qq)**, usando el factor 21.74 (Kg → qq).

El desglose completo (A–Q) replica el panel de la hoja de cálculo original.
